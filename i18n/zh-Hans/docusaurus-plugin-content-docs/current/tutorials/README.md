您将[在本地安装 ***Shifu*** 和相关工具](./demo-install.md)，之后您将[体验 ***Shifu*** 接入设备和与数字孪生交互的过程](./demo-try.md)。

体验 ***Shifu*** 需要您了解 终端 (Terminal) 中的 **命令行操作**。
:::

如果您希望在生产环境安装 ***Shifu***，请查看 [生产环境安装](guides/install-shifu-prod.md)。
